Application ID

What is it?

The application ID of a bot is the ID which is used to invite the bot, and is found on the Discord OAuth invite link button. You most likely do not require to insert one, as it is usually the same as the Client ID, except for bots made before 2016.

Leave this empty if you are unsure

GitHub

What is it?

Visit GitHub

GitHub is a website where you can collaborate and share programming code with people all over the world, using the Git versioning software. In order to be ranked higher in the Discord_Fork website, you should put your code into the GitHub repository, and add a relevant open source licence.
